# Secret Number Guessing Game (Python)

## Overview
This is a simple command-line Python game where:
- One user enters a secret number (hidden input).
- Another user tries to guess the number.
- The player has a maximum of 5 attempts.

## Features
- Secure input using `getpass` (secret number is hidden)
- Limited attempts (5 tries)
- Feedback after each guess
- Displays remaining lives

## Requirements
- Python 3.x

## How to Run

Run the program:
```bash
python "Method.py"
